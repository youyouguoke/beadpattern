Create a modern web application called "BeadPatternAI".

Product Vision

BeadPatternAI is the world's largest searchable library of printable Perler Bead patterns, powered by AI.

This is NOT an AI image generator.

It is a discovery-first platform where users browse thousands of printable bead patterns before generating new ones.

AI generation is a secondary feature that is only used when an existing pattern cannot be found.

The product combines:

• Pinterest-style discovery
• SEO-first pattern library
• Printable PDF downloads
• AI Pattern Generator
• Modern search experience

Target Users

- Perler Bead hobbyists
- Hama Bead creators
- DIY craft lovers
- Parents
- Teachers
- Pinterest users
- Pixel art enthusiasts
